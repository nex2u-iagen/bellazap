# Guia de Configuração LLM (Provedores Externos)

Para garantir que a BellaZap nunca fique offline, você pode configurar provedores de contingência.

## 1. OpenAI (Fallback Padrão)
- **Modelo Recomendado:** gpt-4o-mini ou gpt-3.5-turbo.
- **API Key:** Obtenha em platform.openai.com.

## 2. Google Gemini
- **Modelo Recomendado:** gemini-1.5-flash.
- **API Key:** Obtenha no Google AI Studio (aistudio.google.com).

## 3. Groq (Alta Velocidade)
- **Modelo Recomendado:** llama-3-70b-8192.
- **API Key:** Obtenha em console.groq.com.

---
*Dica: Use o modelo local (Ollama) como prioridade para reduzir custos.*
