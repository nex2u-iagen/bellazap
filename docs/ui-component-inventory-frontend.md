# Frontend UI Component Inventory

This document lists the main UI components found in the frontend application, organized by their feature directories. The components are built using React and Chakra UI.

## Components by Feature

### `frontend/src/features/admin/`
-   **`AdminManagement.tsx`**: Likely handles overall administration tasks and UI.
-   **`PagamentosMonitor.tsx`**: Displays and manages payment-related information for administrators.

### `frontend/src/features/auth/`
-   **`AdminLogin.tsx`**: Provides a login interface specifically for administrators.
-   **`Login.tsx`**: A general-purpose login interface for other users (e.g., resellers).

### `frontend/src/features/revendedora/`
-   **`Financeiro.tsx`**: Manages financial views and functionalities for resellers.
